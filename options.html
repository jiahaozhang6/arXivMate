模型 Profiles

参考 llm-for-zotero 的配置方式：保存多个模型配置；保存后可在论文聊天界面按任务选择使用。

推荐用法

  • 日常速览：选择便宜、速度快的模型，例如 DeepSeek flash 或 gpt-4o-mini。
  • 深读全文：切到长上下文或质量更高的模型，再打开论文面板里的“全文”；DeepSeek/MiniMax 会使用 PDF 文本抽取上下文。
  • 本地实验：保留一个 Ollama profile，不需要 API key。
